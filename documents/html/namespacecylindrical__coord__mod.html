character(len=20), public cylindrical_coord_mod::flx_form = 'xss'

FLX_FORM prescribes the table format of the flux distribution.

  • 1. xss: file format used in SMAP
  • 2. flx: file format used in HINT
  • 3. eqdsk: file format used in EFIT
  • 4. eqdata: file format used in TOPICS

◆ mag_form

character(len=20), public cylindrical_coord_mod::mag_form = ''

MAG_FORM prescribes the table format of the magnetic field.

  • 1. mgrid: file format used in MAKEGRID
  • 2. mgo: file format used in KMAG/KMAG2
  • 3. mag: file format for the equilibrium field used in HINT
  • 4. vac: file format for the vacuum field used in HINT
  • 5. movie: file format used in the MIPS code
  • 6. mips_eq: file format for the equilibrium field used in MIPS
